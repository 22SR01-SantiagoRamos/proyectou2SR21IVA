Investigación de Etiquetas HTML

Etiquetas Básicas

<html>

Define el inicio y fin de un documento HTML.

<head>

Contiene metadatos sobre el documento.

<title>

Define el título de la página web.

<body>

Contiene el contenido visible de la página web.

<p>

Define un párrafo de texto.

Ejmplo de Etiquetas Básicas

Este es un párrafo dentro del <body> de nuestro documento HTML.

Etiquetas de Tablas

<table>

Define el inicio de una tabla.

<tr>

Define una fila dentro de una tabla.

<td>

Define una celda normal en una fila.

<th>

Define una celdade encabezado en una tabla.

Etiquetas de Listas

<ul>

Crea una lista no ordenada.

<ol>

Crea una lista ordenada.

<li>

Define un elemento de una lista.

Ejemplo de Listas

Lista No Ordenada:

  • Manzana
  • Banana
  • Cereza

Lista Ordenada:

  1. Estudiar
  2. Pracicar
  3. Examen

Etiquetas de Formularios

<form>

Define un formulario para recopilar datos.

<input>

Crea un campo de entrada de datos.

<label>

Proporciona una etiqueta para un campo de entrada.

<select>

Crea un menú desplegable.

<option>

Define una opción dentro de un menú desplegable.
